#jquery-xjzTable
一个功能强大且简单易用的表格插件.

![输入图片说明](https://xujunzhou.github.io/jquery-xjzTable/doc/assets/img/img.png "在这里输入图片标题")


特点：
	支持分页、分页信息、分页大小
	支持ajax数据源、本地数据源
	支持排序
	单元格鼠标悬停浮动tooltip
	自定义单元格内容渲染
	精准控制dom生成
	回调函数
	强大的表格刷新方法
	bootstrap样式
	支持自定义css样式类
	配置简单
	兼容IE8
	多语言支持
	...
